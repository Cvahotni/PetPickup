# PetPickup
A small plugin that lets you pickup your pets into spawn eggs.

# Demonstration
- https://www.youtube.com/watch?v=Dh2Ig2oR184

# Installation
- You can find the pre-built binary at https://github.com/Cvahotni/PetPickup/releases
- Currently supports Spigot and Paper versions 1.21+.

# Building
- Clone the repository to your folder
- Open with IntelliJ Idea 2024.3 or any other compatible IDE.
- Create and configure a task for "maven package".
- If the build succeeds, the output files will be in target/

# Permissions
- pp.toggle - Toggle Pet Pickup mode on or off.
- pp.bypass.worldguard - Bypass WorldGuard restrictions.
- pp.bypass.whitelist - Bypass the entity type whitelist.
- pp.bypass.pickup - Bypass pet ownership protections.
